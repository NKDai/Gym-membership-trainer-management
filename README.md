This project develops a Gym Management System using the C programming language. The system is designed to manage gym operations by handling information of both members and trainers.

The program allows users to perform core functionalities such as adding, removing, updating, searching, and sorting member and trainer records. It uses dynamic memory allocation to manage flexible data structures and binary file I/O to ensure data persistence across program executions.

The system is modularized into multiple components including Member, MemberManager, Trainer, TrainerManager, Settings, Sort, and GlobalFunctions, ensuring clear separation of concerns and maintainability.

Additionally, the program supports features such as assigning members to trainers, managing system settings (theme, auto-save, sorting mode), and displaying structured data in formatted tables.

Overall, the project demonstrates fundamental programming concepts including struct-based design, modular programming, dynamic arrays, file handling, and user interaction in C.
